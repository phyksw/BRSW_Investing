학습보다 추론이 더 큰 비용으로 — 토큰값은 폭락, 오픈모델이 가격을 깬다

한 줄 헤드 — 추론·인프라

이제 돈은 학습이 아니라 추론에서 나간다. 배포가 늘면서 추론(서빙)이 컴퓨트 비용의 주류로 — 토큰값은 2년새 10~100배 하락.

지금 — 오픈웨이트(DeepSeek·Qwen·Llama)가 가격을 깨고, 추론 프로바이더(Together·Fireworks·Groq·Cerebras)가 경쟁. 단 추론형(reasoning) 모델은 쿼리당 토큰을 폭증시켜 컴퓨트 수요는 오히려 증가.

워치포인트 — 오픈 vs 클로즈드 점유, 추론 전용 칩(Groq LPU·Cerebras·ASIC), 토큰 단가 바닥, reasoning 토큰 소비 증가율.

동급 성능 기준 토큰 단가 — 2024=100 (정성 추정)

경쟁 + 모델 효율 + 오픈웨이트로 동급 성능 토큰값이 급락. 단가는 떨어지지만 사용량(특히 reasoning)이 폭증해 전체 컴퓨트 지출은 증가 — NVIDIA·HBM에 유리.

관전 포인트

추론은 "AI의 진짜 단가 싸움" — 토큰값 하락은 소비자엔 호재, 모델 랩엔 마진 압박. 하지만 reasoning이 토큰 소비를 키워 컴퓨트 수요는 오히려 증가 → 최대 수혜는 NVIDIA·HBM(SK하이닉스·삼성). 한국 투자자는 추론 스타트업(비상장)보다 곡괭이(반도체·DC) 노출이 깨끗 (→ 메모리 · 컴퓨트).

리스크

① 마진 압박: 토큰값 폭락 — 모델 랩 수익성 악화.
② 오픈모델 잠식: 중국 오픈웨이트가 클로즈드 API 점유 잠식.
③ 추론칩 경쟁: ASIC·LPU가 NVIDIA 추론 점유 일부 가져갈 가능.
